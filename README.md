# Express Easy Routes

Provides an easy to setup routes for an express api based on a folder hierarchy. 
```
/routes
    - index.ts                                      == /
    /test
        - index.ts                                  == /test
        - $id.ts                                    == /test/:id
        - $id.edit.$name.ts                         == /test/:id/edit/:name
```
